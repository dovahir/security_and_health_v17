Modulo de Seguridad y Salud para Odoo V19

Un modulo para Odoo en versión 19, utilizado para llevar un registro de las situaciones de seguridad que puedan ocurrir en la empresa, así como para llevar un registro de salud de los empleados tales como: ● Presión diaria ● Peso, Altura, IMC y Tipo de Sangre ● Análisis Médicos realizados así como sus resultados También permite generar reportes de las situaciones de seguridad creadas y obtener estadisticas de las mismas.
